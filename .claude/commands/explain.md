아래 내용을 분석해줘: $ARGUMENTS

## 이 커맨드가 하는 일

1. 문제 해석
   - 문제를 읽고 어떠한 알고리즘을 적용할 수 있는지 제시
   - 적용 가능한 알고리즘 방식 중 가장 최적의 대안과 그 이유 설명
2. 내가 작성한 코드에서 잘못된 부분 찾기
   - 이렇게 작성하면 안되는 이유 설명
   - 올바른 코드 예시 작성
3. 시간 복잡도, 공간복잡도, 메모리, 제한 조건 등을 고려했을 때 최적의 코드 작성

## 출력 형식

```markdown
# 뮨재 해석

## 적용 가능한 알고리즘

## 최적의 대안

# 내가 작성한 코드 피드백

## 이렇게 작성하면 안되는 이유

## 올바른 작성 예시

# 최적의 코드
```
